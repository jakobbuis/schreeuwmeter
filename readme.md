## Controls

[R] - reset maximum volume reached and current volume <br>
[D] - toggle debug mode (shows individual audio buckets)<br>
[B] - show a prompt to choose the bucket

[number]: set audiosensitivity (default 5):<br>
1: 0.0625<br>
2: 0.125<br>
3: 0.25<br>
4: 0.5<br>
5: 1<br>
6: 2<br>
7: 4<br>
8: 8<br>
9: 16
